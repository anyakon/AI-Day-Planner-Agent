# Tools / APIs

- **Tool: `get_free_slots(date: str)`**
  - **Контракт:** Принимает дату (YYYY-MM-DD), возвращает список свободных окон.
  - **Таймаут:** 3 секунды.
- **Tool: `create_calendar_event(title: str, start: str, end: str)`**
  - **Контракт:** Создает событие. Возвращает ссылку на событие.
  - **Side effects:** Изменяет календарь пользователя.
  - **Защита:** 
    - Строгая Pydantic валидация (start должен быть до end).
    - Идемпотентность: передается хэш `title+start` как idempotency_key, чтобы при ретраях агента не создать 5 одинаковых встреч.

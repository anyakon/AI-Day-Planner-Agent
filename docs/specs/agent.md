# Agent Orchestrator

- **Архитектура:** ReAct (Reason + Act) цикл, реализованный через LangGraph.
- **Stop Condition:** Максимум 10 итераций (шагов) в графе на один запрос пользователя (предохранитель от бесконечных циклов).
- **Правила переходов:**
  - `LLM` -> `ToolNode` (если есть tool_calls)
  - `LLM` -> `ValidatorNode` (если LLM решила, что закончила)
  - `ValidatorNode` -> `END` (если валидация успешна)
  - `ValidatorNode` -> `LLM` (если найдены накладки во времени - инжектим системную ошибку в контекст).
